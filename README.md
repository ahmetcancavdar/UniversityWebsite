# Eskişehir Technical University (ESTÜ) Website Redesign

This is a responsive, static website redesign project for Eskişehir Technical University (ESTÜ). Built using pure HTML and CSS, the project includes various sections detailing the university's academic offerings, campus life, student resources, and official documents.

## Features

- **Multi-page Architecture**: Includes dedicated pages for Home, University, Academics, Students, Campus Life, Documents, and Login.
- **Pure CSS Styling**: Custom and responsive design created entirely with CSS without the use of external UI frameworks.
- **Document Management**: A structured documents page (`documents.html`) categorizing official files into Petitions, Transfer Forms, and Internship Documents with downloadable Word and Excel files.
- **Modern Interface**: Clean and user-friendly web interface inspired by ESTÜ's brand identity.

## Technologies Used

- **HTML5**: For page structure and semantic content.
- **CSS3**: For styling, layout, and responsive design (`style.css`).

## Project Structure

```text
├── academics.html     # Academics section page
├── campuslife.html    # Campus life and facilities page
├── documents.html     # Official documents download page
├── home.html          # Main landing page
├── login.html         # User/Student login interfaces
├── students.html      # Information page for students
├── university.html    # General information about the university
├── style.css          # Main stylesheet for the entire website
├── images/            # Directory containing all visual assets
└── documents/         # Directory for downloadable files (Word, Excel)
```

## How to Run

Since this is a static website, no build steps or local server are required to view the pages.

1. Clone or download the repository to your local machine:
   ```bash
   git clone <repository-url>
   ```
2. Navigate to the project folder.
3. Open `home.html` in your preferred web browser (Google Chrome, Mozilla Firefox, Safari, Edge, etc.) to start exploring the website.
